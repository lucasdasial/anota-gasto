# Anotagasto

**Read in:** [English](#english) · [Português](#português)

---

## English

Ever reached the end of the month wondering where all your money went? Anotagasto solves that.

The app was built for two simple actions:

1. **Log expenses quickly** — as little friction as possible at the moment of spending
2. **Understand where money went** — metrics by category, monthly totals and comparisons

No spreadsheets, no complex apps. You log it, the app organizes it.

### Development

```bash
docker-compose up -d   # start PostgreSQL
mix setup              # install dependencies and create the database
mix phx.server         # start the server at http://localhost:4000
```

## Português

Você já chegou no fim do mês sem saber onde o dinheiro foi? Anotagasto resolve isso.

O app foi criado para duas ações simples:

1. **Registrar gastos rapidamente** — o menor atrito possível no momento do gasto
2. **Entender onde o dinheiro foi** — métricas por categoria, totais mensais e comparações

Nada de planilhas, nada de apps complexos. Você anota, o app organiza.

### Desenvolvimento

```bash
docker-compose up -d   # inicia o PostgreSQL
mix setup              # instala dependências e cria o banco
mix phx.server         # inicia o servidor em http://localhost:4000
```
